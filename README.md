# 🎮 Video Game Sales & Engagement Dashboard

## 🚀 Overview
This project is a data analytics dashboard built using Python, Streamlit, SQL, and Machine Learning to analyze video game sales and user engagement.

---

## 🎯 Objective
- Understand game sales trends  
- Analyze user behavior (ratings, wishlist, plays)  
- Find top genres, platforms, and publishers  

---

## 📊 Features
- Interactive dashboard (Power BI style)  
- Drill-down filters  
- SQL analysis (15 queries)  
- 20+ charts  
- ML forecasting  

---

## 📂 Dataset
- games.csv → user engagement  
- vgsales.csv → sales  

---

## 🛠 Tech Stack
- Python  
- Streamlit  
- Pandas  
- Plotly  
- SQL (SQLite)  
- Machine Learning  

---

pip install -r requirements.txt
